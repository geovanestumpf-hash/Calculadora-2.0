# Calculadora 2.0
Projeto Lógica de Programação
O arquivo contendo o código da calculadora é calculadora.py
O programa solicitará ao usuário seu nome e dois números para realizar as operações matemáticas básicas;
Todo o código é acompanhado de orientações ao usuário final, para facilitar a interação;
O projeto também conta com um script de automação em shell script, o arquivo calculadora.sh;
Este arquivo solicitará acesso como super usuário para atualizar e instalar o phyton3 na máquina;
Após a instalação será executado o arquivo em phyton;
